# Clip-to-Brain 默认 Obsidian 知识库

粘贴链接后，笔记会写入对应子目录。可在 Obsidian 中打开本文件夹作为 Vault。

## 目录

| 路径 | 用途 |
|------|------|
| `03-benchmark-accounts/VideoNotes/` | 对标视频/图文笔记 |
| `02-industry-resources/` | 公众号、行业文章 |
| `04-viral-topics/_inbox/` | 自动生成的选题卡 |
| `04-viral-topics/_template.md` | 选题模板 |

## 使用

```powershell
cd content-archiver-skill
.\clip.ps1 "https://weixin.qq.com/sph/..."
```

入库后在本库搜索标题或打开 `_content_index.md`。
